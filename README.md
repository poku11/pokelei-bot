# Vinted Scanner (Streamlit)

Prototype: scanner d'annonces Vinted, scoring achat/revente, UI Streamlit.

## Install
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

## Run locally
streamlit run vinted_app.py

## Deploy
- Pousser sur GitHub.
- Déployer sur Streamlit Cloud (nouveau app → link repo).
- Ou Docker + container hosting.

## Important
Respecte les Terms of Service de Vinted. Ne pas automatiser achats sans accord. Mode prototype.
